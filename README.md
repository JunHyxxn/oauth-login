## 기술 스택

---

### Front End

- Axios ^1.4.0
- React ^18.2.0
- React-dom ^18.2.0
- react-router-dom ^6.14.2
- recoil ^0.7.7
  recoil-persist ^5.1.0

### Back End

- Java 11
- Spring Boot 2.7.13
- Spring Security
- OAuth2 Client
- JWT
- Spring Data JPA
- QueryDSL
- MySQL
- Redis
